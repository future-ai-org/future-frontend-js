## lilit.ai frontend

<br>

#### frontend for <a href="https://lilit.ai">lilit.ai</a>, deployed on vercel

<br>

---

### running locally

<br>

install dependencies:

```bash
make install
```

<br>

build and start local server:

```
make server
```

<br>

---

### pushing new dev changes

<br>

make sure to lint the code with:

```bash
make lint
make format
```
